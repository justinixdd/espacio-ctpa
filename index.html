🌐 Diseño de Software (10°)

En este planeta, aprenderás los fundamentos del diseño de software, incluyendo diagramas UML, patrones de diseño y arquitectura de sistemas. Es la base para construir aplicaciones robustas y escalables.

  • Introducción a la programación
  • Lógica de programación
  • Diseño de bases de datos
  • Principios SOLID
×

💻 Desarrollo Frontend (11°)

Explora el arte de crear interfaces de usuario interactivas y atractivas. Dominarás HTML, CSS y JavaScript, y te sumergirás en frameworks modernos como React o Vue.js.

  • HTML5 y CSS3 avanzado
  • JavaScript moderno (ES6+)
  • Frameworks y librerías (React, Vue, Angular)
  • Diseño responsivo y UX/UI
×

⚙️ Desarrollo Backend (12°)

Adéntrate en el lado del servidor, donde la lógica de negocio y la gestión de datos cobran vida. Aprenderás sobre bases de datos, APIs RESTful y lenguajes como Node.js, Python o PHP.

  • Gestión de bases de datos (SQL, NoSQL)
  • Desarrollo de APIs RESTful
  • Lenguajes de servidor (Node.js, Python, PHP)
  • Seguridad web y autenticación
